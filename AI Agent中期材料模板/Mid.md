# 中期

【Agent训练营中期材料提交要求】

为及时了解各项目组当前进展，请各项目组按要求提交中期材料。本次中期检查不安排现场汇报，主要采取材料函审的形式进行。

请各项目组围绕已开发的 Agent Demo，简要说明项目内容、系统设计、当前完成情况、运行效果及后续计划。提交材料应结构清晰、内容完整，能够较好体现项目进展和阶段性成果，便于评审老师审阅。评审的优秀项目有机会获得追加的训练资源。

请各组于6月9日23：59前将材料打包提交到FudanAICS@163.com。谢谢大家配合！

一、提交内容

请提交以下材料：

1. 项目说明文档：说明项目目标、Agent Demo 功能、系统设计、当前进展和后续计划；
2. Demo演示视频：展示 Agent 的主要功能和运行效果；
3. Demo演示说明（可选）：如有需要说明的内容建议视频内标注或语音说明，如有需要可以附加说明材料；
4. 项目代码或代码仓库链接；
5. 代码运行说明：包括运行依赖、安装流程等
6. 其他补充材料：测试样例等，可自行选择提交。

二、提交格式

请将材料打包提交，邮件主题及附件压缩包的命名格式均为：

Agent训练营中期材料_项目名称_小组名称.zip

压缩包结构建议如下：

Agent训练营中期材料_项目名称_小组名称/
├── 01_项目说明书.pdf
├── 02_demo_video.mp4
├── 03_Demo视频说明.md / .docx / .pdf（可选）
├── 04_src/ 或 代码仓库链接.txt
└── 05_代码运行说明.md / .docx / .pdf
