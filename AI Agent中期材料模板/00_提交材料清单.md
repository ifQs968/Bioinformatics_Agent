# AI Agent 中期材料提交清单


## 一、提交文件结构

```text
AI Agent中期材料/
+-- 00_提交材料清单.md
+-- 01_项目说明文档.md / .docx / .pdf
+-- 02_Demo视频说明.md / .docx / .pdf
+-- 03_代码与补充材料说明.md / .docx / .pdf
+-- 04_demo_video.mp4
+-- 05_src/
|   +-- 项目代码
+-- 或05_代码仓库链接.txt
```

## 二、必交材料

| 材料  | 说明 |
|---|---|
| 01_项目说明文档 | 说明项目目标、Agent 功能、系统设计、Demo 运行说明、当前进展与后续计划 |
| 02_Demo视频说明  | 说明截图或视频对应的 Demo 流程和关键结果 |
| 03_代码与补充材料说明 | 说明代码结构、核心文件、运行入口、补充材料与限制 |
| 04_demo_video.mp4  | 至少提供截图或视频中的一种；建议二者都提供 |
| 05_src/ 或 代码仓库链接.txt  | 可提交源码目录，也可提供可访问的仓库链接 |

## 三、提交前检查

- [ ] 所有文档中的项目名称、成员、联系方式已填写完整。
- [ ] Demo 启动命令经过实际验证。
- [ ] 依赖安装步骤可由他人复现。
- [ ] API Key、账号密码、私有 Token 等敏感信息已删除或替换为占位符。
- [ ] 截图或视频能清楚展示输入、Agent 执行过程和最终输出。
- [ ] 代码仓库链接可访问，或 `src/` 中已包含必要源码。
- [ ] 文档已导出为要求格式，建议 `.pdf`。
- [ ] 文件命名与提交要求一致。
